# zapp
Simple http server with websocket powered live reloading.

![](http://zapp.wor.io/zapp.png)

## Usage

    $ mkdir www
    $ cd www
    $ touch index.html
    $ npm install -g zapp
    $ zapp
  
Direct your browser at [http://localhost:80/](http://localhost:80/). Start editing index.html.
